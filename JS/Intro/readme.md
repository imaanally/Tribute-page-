## variables

### rules for creating variables 

1. spaces not allowed 
2. only letters digits underscore and dollar signs are permitted 
3. case matters when it comes to variables 
4. a letter undescore or dollar should be the first letter no matter what, never start with a number 
5. words reserved for JS should not be used as a variables
